This content is as described in my thesis, "Neural Decoding of Imagined Music: A Machine Learning Framework for EEG-Based Audio Feature Reconstruction." It includes the code used to train models and generated MIDI, the original MIDI files that features were extracted from, the final generated MIDI, and survey responses. In the original_midi file, there are two versions of each stimulus that are times slightly different tempos. This is due the tempo being slightly changed halfway through participants in the original OpenMIIR study that the MIDI is derived from.
